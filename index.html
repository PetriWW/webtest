What is this? How does it work?

Mapperino is a collaboration program built in Electron and Python. A tool for sorting out local beatmaps based on similarities.

This operates by parsing local beatmaps and analyzing various factors of a beatmap, this data is then used for calculations and the results are presented to the user. The results can be copied to clipboard and pasted in-game, because currently we do not have an API for opening them directly on Osu. It's still WIP.

Essentially the program helps you to find more standard maps that feel the same or have similar characteristics.

Step 1


Download the latest version here and unpack the file. The next step requires online connection.

Step 2


Open the application and complete authorization. This process is handled by OAuth 2.0 provided by Osu!

Step 3


The program will ask the user to initiate the 'Parse Beatmaps' process. Once the beatmaps are loaded, users can search for a specific beatmap, and the program will then present the results.

Frequently Asked Questions



Q: Can I download beatmaps with this?
A: No, this tool works only with local beatmaps.

Q: Does this work for all game modes?
A: No, Mapperino currently works only for standard mode beatmaps.

Q: Why do I need to authorize?
A: Some of the functions work by using OAuth 2.0, which requires users to authorize.

Q: Where can I get help?
A: Help requests, bug reports, suggestions etc can be done on our Discord server.

Q: Does this work with lazer?
A: No, the app is designed to work with beatmaps from stable. But you can use the results for lazer.
