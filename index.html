Project Mapperino

Beatmap assistant for Osu!

Modern light-weight desktop client for assisting with beatmap search. A tool that sorts out your beatmaps based on similarity.

Want to find beatmaps that feel like the one you just played? Try out the tool now!

What is this? How does it work?

Mapperino is a collaboration program built in Electron, made by people who have wished for features that didn't exist. The intended purpose of this application is to sort beatmaps based on their similarities. The idea was inspired by an older abandoned project that aimed to do the same thing, except this is a more accurate revision of it.

Osu beatmaps are files containing timing points and various note types distributed over the duration of a song, including other factors. This can be utilized in calculations to find, match, and display results based on the similarity of specific factors using JS Math. Yada yada.

Essentially the program helps you to find more maps that feel the same.

Step 1


Download the latest version here and proceed with the installation. The app requires an internet connection for the next step.

Step 2


Open the application and complete authorization. This process is handled by OAuth 2.0 provided by Osu!

Step 3


The program will prompt for the game location and proceed to search for beatmaps. From there, you can select a beatmap, and the program will display results.
